HTML Tags

  1. <html>: The root element of an HTML document.
  2. <head>: Contains meta-information about the document, such as the title and links to stylesheets.
  3. <title>: Sets the title of the document, shown in the browser's title bar or tab.
  4. <meta>: Defines metadata about the HTML document, such as character set and viewport settings.
  5. <link>: Used to link external resources, such as stylesheets.
  6. <style>: Contains internal CSS styles for the document.
  7. <body>: Contains the content of the HTML document, such as text, images, and other elements.
  8. <h1> to <h6>: Heading tags, with <h1> being the highest level and <h6> the lowest.
  9. <p>: Defines a paragraph of text.
  10. <a>: Defines a hyperlink, which can link to another page or resource.
  11. <img>: Embeds an image in the document.
  12. <ul>: Defines an unordered list.
  13. <ol>: Defines an ordered list.
  14. <li>: Defines a list item, used inside <ul> or <ol>.
  15. <div>: Defines a division or section, often used for layout purposes.
  16. <span>: Defines a span of text, often used for styling purposes.
  17. <form>: Defines a form for user input.
  18. <input>: Defines an input field within a form.
  19. <button>: Defines a clickable button.
  20. <table>: Defines a table.
  21. <tr>: Defines a table row.
  22. <th>: Defines a table header cell.
  23. <td>: Defines a table data cell.
  24. <header>: Defines a header section for a document or section.
  25. <footer>: Defines a footer section for a document or section.
  26. <nav>: Defines a navigation section, containing links to other parts of the document or site.
  27. <article>: Defines an article or independent piece of content.
  28. <section>: Defines a section of content within a document.
  29. <aside>: Defines content aside from the main content, such as a sidebar.
  30. <main>: Defines the main content of a document.
